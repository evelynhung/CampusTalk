「為開放的 Web 而打造的作業系統」

Firefox OS 是一個新的行動裝置作業系統，由 Mozilla 的 Boot to Gecko (B2G) 專案開發。它使用 Linux 核心，開機之後進入以 Gecko 為基礎的執行引擎（Runtime Engine）讓用戶執行完全透過 HTML、JavaScript 及其他開放網路程式 API（Open Web Applicaiton APIs）所開發的程式。

Mozilla 認為...

  1. 使用者應該有更多的選擇
    • 不想選擇 ≠ 願意失去選擇
  2. 填補行動市場的空缺
    • 低階的硬體上媲美 Android 中階硬體的智慧型手機體驗
  3. App 開發者不該受任何平台廠商的控制
    • 以開放的 Web API 開發應用，達到真正的跨平台
  4. 真正的 open 是眾人皆可參與的完全開放計畫
    • 公開開發中的原始碼、會議討論

致力於標準化

  • 新的 web APIs: 用網頁技術直接操控硬體(電話、簡訊、相機、USB、藍牙...)的能力
  • 權限控管機制: 確保新的操控能力對所有 web 應用是安全的
  • 開機驅動: 對行動裝置(手機、平板)有最基本的開機驅動方式
  • 應用(app): 一組內建能夠展示系統基本功能的應用
